# wspr-tx

Transmit WSPR signal with FL2K USB converter
============================================

Uses WSPR 4FSK encoding routine encode.py by SM0YSR 
https://github.com/robertostling/wspr-tools

Dependencies:
=============
* python 3.7
* numpy
* fl2k_tcp

Usage:
======
* Make sure to edit wspr_config.py
* python3 wsprtx.py
